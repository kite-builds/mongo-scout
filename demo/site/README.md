# Hosted demo page

Source for the public demo at **https://mongo-scout-demo.surge.sh** — an
HTML5 player for the narrated screencast plus the one-command repro steps
and repo link.

Deploy (media lives in `../media/`):

```bash
cp ../media/mongo-scout-demo.mp4 ../media/mongo-scout-demo.gif .
surge ./ mongo-scout-demo.surge.sh
```
